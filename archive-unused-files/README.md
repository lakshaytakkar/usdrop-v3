# Archive - Unused/Temporary Files

This folder contains files that have been archived as they are no longer needed for the current working project.

- `snap_miin3sjp1o36majndf.csv` - Source CSV data file that was used to generate product data. The data has been processed and is now in `src/data/product-picks.ts`. If you need to reprocess, update the path in `scripts/process-csv-to-products.ts`.

### blender-files/

### after-effects-files/
- All `.aep` files from `public/Multimedia animated icons/` - After Effects project files. The exported JSON animation files remain in the project.

### duplicate-scripts/
- `process-csv-to-products.js` - Duplicate JavaScript version of the TypeScript script. The TypeScript version (`process-csv-to-products.ts`) is the one being used.

### test-images/
- `labubu*.jpg` and `labubu*.png` - Test/development images that are not referenced in the codebase.

## Note
All active project files remain untouched. These archived files can be safely deleted if you're certain they're no longer needed, or kept for reference.
