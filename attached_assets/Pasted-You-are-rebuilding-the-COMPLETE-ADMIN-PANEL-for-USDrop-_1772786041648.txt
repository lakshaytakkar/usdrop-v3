THE COMPLETE USER JOURNEY (context for admin)
════════════════════════════════════════════

PRE-SALES JOURNEY (what admin tracks in Pipeline):
- User discovers Suprans via Instagram/events/referrals → lands on suprans.in
- Signs up for free on USDrop.ai → profile created, lead record auto-created with stage = new_lead
- Explores platform → activity logged (page views, feature clicks) → lead_scores.engagement_level moves cold → warm
- Watches free onboarding chapters (onboarding_videos) → each completion logged in onboarding_progress
- Completes ALL free chapters → lead_scores.auto_stage flips to 'hot' automatically → admin gets badge alert
- Admin sees hot lead → reviews their activity, chapter completions, last active date
- Admin sends payment link (₹29,999 for Mr. Suprans mentorship) → payment_links record created
- Admin calls lead or lead books call → outcome logged in notes
- Lead pays → admin confirms payment → lead stage moves to 'converted' → client journey begins

POST-SALES JOURNEY (what admin tracks in Clients):
- Admin assigns client to next rolling batch (new batch every 2 weeks) → batch_members record created
- Welcome WhatsApp message copied and sent manually by admin
- Client starts Week 1-2: fundamentals + product research modules → module_completions tracked
- Week 1-2 completion → product explorer + video library features unlock automatically
- Client starts Week 3-4: store setup begins → admin introduces LLC upsell contextually
- LLC need becomes obvious (US marketplaces require US entity) → client purchases LLC package
- Admin creates llc_applications record → LLC journey begins in parallel
- Week 3-4 completion → ad creative tools + meta ads library unlock
- LLC milestones updated manually by admin: filed → EIN → BOI → bank setup → Stripe setup → complete
- LLC complete → full platform access unlocked (including order fulfillment tools)
- Week 5-6: ads + marketing phase → client runs first ads with USDrop tools
- Week 7-8: fulfillment + scaling phase → first orders managed
- Graduation: admin marks client complete → certificate issued → alumni status
- Post-graduation: alumni community access, optional advanced cohort upsell

SUPPORT JOURNEY (parallel to both):
- Client raises ticket from /dashboard/support at any point
- Admin triages: assigns type (technical/llc/course/shopify/payment/escalation) and priority
- Admin or rep replies → client notified
- Complex issues escalated to Lakshay
- Ticket resolved → archived after 30 days
